Turn‑Based Battle System (EmberRed)

Stack: Unreal Engine / C++ / PaperZD

EmberRed is a Pokémon-inspired project where I implemented a turn-based battle system in Unreal Engine. It features data-driven abilities, AI trainers, and smooth 2D animations using the PaperZD plugin.

  • Battle manager controlling phases (select → resolve → animate)
  • Enhanced Input for trainer & menu navigation
  • Data assets for moves, items, and trainer rosters
