Magic Mayhem (Co-op Wave Survival)

Stack: Unity / C# / Multiplayer

Magic Mayhem is a co-op wave survival game built in Unity. I implemented scalable wave spawning, player abilities with cooldowns and VFX, and integrated sound and UI systems to enhance immersion.

  • Wave system supporting progressive difficulty and varied enemy types
  • Implemented abilities: Ice Block, Chain Lightning, Meteor Strike, Ice Cone
  • Collaborated with a 4-person team using Git for version control
